Message Center

  • Preferred Name Display

    We are pleased to announce that the CIS Portal will now display a preferred name in the Profile tile if you have set one. The Preferred Name can be set by clicking the Update Profile bar at the bottom of the tile to open the Student Profile application, then editing the Preferred Name option. If you have not set a preferred name, the profile tile will still default to your legal name. Note, the preferred name only displays in the Profile tile. It is not set or displayed in other systems that you may access through CIS.

  • New CIS Portal

    Welcome to the new CIS Portal. You'll notice that the new CIS uses iconized tiles to provide access to University systems and resources. These tiles are delivered through a series of customizable home pages that are dedicated to user roles, functional areas, and common themes. To get familiar with the new navigation and system features, refer to the CIS Portal Getting Started Guide. For more detailed information on the system, refer to the CIS Portal Knowledge Base.

    We hope you enjoy using the new system. If you have thoughts about the system or suggestions to help us make it better, please provide your feedback.

While messages can be sent many ways - such as email and social media - information sent by text directly to cell phones is the fastest and most reliable.

I opt out of this text alert system. What will that mean?


The campus alert system notifies students, faculty and staff of serious emergencies. These messages are sent using a combination of email and text messages. Opting out of receiving these messages via text means you might not receive critical safety information in a timely manner. The university encourages everyone to provide all available contact information - including cell phone numbers - for use in these emergency situations. Numbers will not be used for non-emergency purposes, like marketing or solicitation. For more information on emergency alerts, please see our FAQ page.
